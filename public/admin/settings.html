Receive real UPI payments

When you fill this in, buyers who choose GPay/PhonePe/Paytm/UPI at checkout will see your real UPI ID and QR code and pay you directly. There's no gateway involved, so payments are self-reported by the buyer — review them in Orders.

Real email notifications

Password reset codes and "new software" alerts are sent as real email automatically, as long as the server has SMTP credentials configured in its .env file. This isn't set here in the admin panel — see the README for setup (any provider works: Gmail with an App Password, Resend, Brevo, SendGrid SMTP, etc). Without it, password reset still works by showing the code on screen instead of emailing it.

Danger zone

This affects every visitor to the site, not just your browser — resetting data on a real shared server isn't something to do casually. See the README for how to do it safely against MongoDB directly.
